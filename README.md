***

#### Getting started

1. Clone the repo

    ```sh
    $ git clone git@github.com:dankreiger/login-form.git
    ```

2. `cd` into project

    ```sh
    $ cd login-form
    ```

3. Install dependencies (yarn or npm)

  - **yarn**:

    ```sh
    $ yarn
    ```

  - **npm**:

    ```sh
    $ npm install
    ```  

4. Start server (yarn or npm)

  - **yarn**:

    ```sh
    $ yarn start
    ```

  - **npm**:

    ```sh
    $ npm start
    ```  

5. http://localhost:3000/ will open automatically
